DigitalOcean + Dokku + Meteor

So for two weeks now I’ve been running Dokku on DigitalOcean for my Meteor apps (and some old PHP apps as well). I’ve read a few guides about it, but I feel they all leave something out so I’ll try to collect everything I’ve learned so far about the subject. What is Dokku? Docker powered mini-Heroku. The smallest PaaS implementation you’ve ever seen. Run docker containers without knowing Docker, push code via git and support for buildpacks.
